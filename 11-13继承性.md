# 层叠性： 权重就近 行内 

![](/assets/cengdie.png)

```
p {
           color:red;
       }

        .xixi {
            color: blue;
        }
        #haha {
            color: yellow;
        }
    </style>
</head>
<body>
    <div>
        <p id="haha" class="xixi">haha1</p>
        <p>haha2</p>
        <p>haha3</p>
```
