# 定义列表

### dl: definition list   定义列表

### dl:definition title   定义标题

### dd:definition description 定义标题的描述

### 一个标题dt可以有多个dd描述

```
<dl>
        <dt>北京</dt>
        <dd>中国的首都</dd>
        <dt>上海</dt>
        <dd>东方明珠塔</dd>
        <dt>深圳</dt>
        <dd>购物中心</dd>
    </dl>
```

![](/assets/definitionlist.png)

### dd里面也可以像ul那样嵌套别的标签

```
<dl>
        <dt>北京</dt>
        <dd>
            <a href="#">haha1</a>
            <a href="#">haha2</a>
            <a href="#">haha3</a>
            <a href="#">haha4</a>
        </dd>
```
